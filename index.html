About

I am a full stack software engineer at Cardstack, where I am working on tools that help non-developers to build their own products on the web. I'm on the core team of a front-end JavaScript framework called Ember.js, where I help maintain repositories, write new content, and serve in a leadership capacity.

I enjoy rapid prototyping, JavaScript, CSS, HTML, video tutorial shooting and editing, mockups, user testing, and API design. I aiom to create accessible web apps and build inclusive, healthy engineering teams.
